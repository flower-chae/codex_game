# Run and Gun Prototype

Phaser 3 기반의 하강형 슈팅 샘플입니다.
모바일 세로 화면(포트레이트) 중심으로 맞췄고, 터치 조작을 지원합니다.

## 실행
1. `run_and_gun/index.html` 열기
2. 또는 로컬 서버 실행
   - 예: `python -m http.server 5500`
   - 접속: `http://localhost:5500/run_and_gun/`

## 조작
- 모바일 이동: 화면 좌/우 터치 유지로 이동 (터치 위치 변경 시 방향 전환)
- 공격: 자동 연사 (버튼 입력 불필요)
- 모바일 재시작: 화면 우상단 `RESTART`
- 키보드: `Left`, `Right`
- 재시작: `R`

## 규칙
- 적은 화면 위에서 도로 안으로 내려옵니다.
- 적은 색상/종류별로 HP가 다르며, 중반 이후 `고체력 탱커`와 `고속 러너`가 등장합니다.
- 아이템은 간헐적으로만 드랍됩니다.
- 아이템 색상은 하나로 통일되어, 획득 전에는 효과를 알 수 없습니다.
- 무기 레벨이 2 이상이면, 5초마다 1단계씩 자동으로 감소합니다.
- 시작 HP는 `3`입니다.
- 적과 `3번 충돌`하거나, 적을 못 막아 `3번 돌파`시키면 게임오버입니다. (충돌/돌파 모두 HP 1 차감)
