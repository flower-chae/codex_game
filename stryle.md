# Run-and-Gun Style (하강형 슈팅)

## 장르 정의
- `고정 화면 하강형 슈팅(arcade shooter)` 스타일
- 플레이어는 화면 하단을 좌우로 이동
- 적은 화면 상단에서 하단으로 내려옴
- 플레이 목표는 `생존 + 처치 점수 + 무기 강화`

## 플레이 코어 루프
1. 좌우 회피 + Space 홀드 연사
2. 적 격추 후 파워업 획득
3. 무기 레벨 상승으로 화력 증가
4. 더 많은 적/패턴을 처리하며 생존 시간 연장

## 조작/입력 감각
- 이동: `Left`, `Right`
- 사격: `Space` (누르고 있으면 자동 연사)
- 재시작: `R`
- 입력 응답은 즉각적이어야 함 (지연감 최소화)
- 사격 쿨다운은 짧게, 레벨업 시 더 짧게

## 전투 스타일
- 총알에 맞은 일반 적은 즉시 제거 (`원샷 킬`)
- 무기 강화 시 아래 순서로 체감 강화
  - 연사 속도 증가
  - 탄속 증가
  - 탄 개수/각도 확장(2-way, 3-way)
- 파워업은 리스크-리턴 구조
  - 화면에 잠시만 유지
  - 먹으러 가면 피격 위험 증가

## 적 디자인 원칙
- 기본 적: 빠르게 떨어지는 저체력
- 변형 적: 느리지만 더 큰 판정/점수
- 혼합 스폰으로 리듬 형성
  - 단독 낙하
  - 양옆 동시 낙하
  - 중앙 집중 낙하
- 패턴 난이도는 `시간 기반`으로 점진 상승

## 레벨/웨이브 연출
- 20~30초 단위 미니 웨이브 추천
- 웨이브 말미에 밀도 급상승(짧은 피크)
- 피크 직후 2~3초 숨 고르기 구간 제공
- 보상 타이밍은 피크 직후 배치

## 비주얼 방향
- 픽셀 아케이드 느낌 유지 (명확한 실루엣)
- 배경은 어둡게, 탄/적은 고대비로 강조
- 피격 피드백 필수
  - 적: 즉시 소멸 + 짧은 플래시
  - 플레이어: 짧은 틴트/무적 프레임(선택)
- UI는 상단 고정
  - `Score`, `HP`, `Weapon Lv`, 상태 메시지

## 밸런스 기준 (초기값 가이드)
- 시작 HP: `3`
- 적 스폰 간격: `0.3s ~ 0.7s` 범위 랜덤
- 파워업 드랍률: `15% ~ 20%`
- 무기 최대 레벨: `4`
- 실패 조건:
  - 적 충돌로 HP 소진
  - 적이 바닥 도달 시 HP 감소

## 한 줄 스타일 요약
- `짧은 템포 + 높은 가시성 + 즉시 피드백`의 아케이드 하강형 슈팅.
